# Sitio Web con Express

Este proyecto consiste en servir un sitio web estático utilizando Express.js y Node.js.

## Tecnologías usadas
- Node.js
- Express.js
- HTML, CSS, JavaScript
- Bootstrap

## Instalación
1. Clonar el repositorio
2. Ejecutar:
   npm install
3. Iniciar el servidor:
   node server.js

## Rutas disponibles
- / → Página principal
- /about → Página Acerca de
- /contact → Página de contacto
-/shop→ vizualiza los productos


## Servidor
El servidor se ejecuta en:
http://localhost:3000
